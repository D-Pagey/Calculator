JavaScript Calculator

This is an Advanced Front End Developer project from the FreeCodeCamp curriculum. The user stories to be completed are:

  • I can add, subtract, multiply and divide two numbers.
  • I can clear the input field with a clear button.
  • I can keep chaining mathematical operations together until I hit the equal button, and the calculator will tell me the correct output.

This was made using: Flexbox, SCSS, and native JS. No CSS or JavaScript frameworks or libraries were used. The code can be seen on Github.
